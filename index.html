Latest Projects

Cotton Aphid detection using AI ML

New
Jupyter Notebook Code

Used latest AI/ML techniques for custom object detection which can provide valuable information to farmers about most hazardous pests found over regional crops.

Developed and trained the AI/ML model using tensorflow object detection and other Deep Neural Network techniques to identify/predict probable harmful pests generally found in regional crops like Cotton and Citrus[Oranges].

This model will try to recognise certain selected pests over the images and its predicted value along with creating bounding boxes.

It can be used as an API to predict these custom objects[i.e. pests] for any given images, videos or live webcam feed. This API can be used further in any web/mobile application.

Trained on a dataset of ~2000 images of most harmful pests which generally cause damages in regional farms, such as Cotton Boll Weevil, Cotton Aphid and Citrus Canker.

Idea is to predict pests over any media files with its own description and it is supposed to be benificial to farmers by allowing them to make better decisions about which pesticides to be choosed.
